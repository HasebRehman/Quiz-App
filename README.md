# Quiz Web Application

A dynamic and interactive Quiz Web Application developed using HTML, CSS, and JavaScript.
This app fetches quiz questions and options dynamically and displays them in a clean, user-friendly interface.

---

## 🚀 Features

- Dynamically fetched quiz questions
- Multiple-choice options
- Real-time question rendering
- Clean and responsive user interface
- Smooth user interaction

---

## 🛠️ Technologies Used

- **HTML** – For structuring the quiz layout  
- **CSS** – For styling and responsive design  
- **JavaScript** – For fetching data, handling logic, and user interaction

## ▶️ How to Use This Project

Follow these steps to run the Quiz App locally:

### 1️⃣ Open Terminal / Command Prompt

- **Windows:** Press `Win + R`, type `cmd` or open **PowerShell**, and press Enter  
- **Mac / Linux:** Open **Terminal**

---

### 2️⃣ Navigate to the Folder Where You Want the Code

Choose a location where you want to download the project. For example, to use Desktop:

```bash
cd Desktop
```

> Replace `Desktop` with any folder path you prefer.

---

### 3️⃣ Clone the Repository

Run the following command to download the project from GitHub:

```bash
git clone https://github.com/HasebRehman/Quiz-App.git
```

This will create a folder named `Quiz-App` with all the project files.

---

### 4️⃣ Navigate to the Project Folder

```bash
cd Quiz-App
```

---

### 5️⃣ Run the Application

- Open the `index.html` file in **any modern web browser** (Chrome, Edge, Firefox)  
  **OR**  
- Simply double-click the `index.html` file  

The Quiz App will start immediately ✅

---

### 6️⃣ Alternative Method (Without Git)

- Go to the GitHub repository: [https://github.com/HasebRehman/Quiz-App](https://github.com/HasebRehman/Quiz-App)  
- Click **Code → Download ZIP**  
- Extract the ZIP file to any folder  
- Open `index.html` in your browser
